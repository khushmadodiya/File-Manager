# Device Connection Guide

This application connects your Android device to macOS using a secure, developer-supported method.  
Please follow the steps below carefully. This is a **one-time setup** per device.

---

## Requirements

- Android phone (Android 7.0 or higher recommended)
- USB cable (data cable, not charge-only)
- macOS system
- Device unlocked during first connection

---

## Step-by-Step Connection Process

### 1. Connect Your Device via USB
- Plug your Android phone into your Mac using a USB cable
- Unlock your phone

---

### 2. Enable Developer Options

Developer Options must be enabled to allow communication.

**Steps:**
1. Open **Settings** on your Android device
2. Go to **About phone**

### 3. Enable USB Debugging

USB Debugging allows your device to communicate with your Mac.

**Steps:**
1. Open **Settings**
2. Go to **Developer options**
3. Enable **USB debugging**
4. Confirm the dialog when prompted

In some cases, file access requires File Transfer mode.

## Troubleshooting

### Device Not Detected?
- Ensure USB Debugging is enabled
- Reconnect the USB cable
- Try a different USB port or cable
- Make sure the device is unlocked

### Permission Dialog Not Showing?
- Disable and re-enable USB Debugging
- Reconnect the device
- Restart the phone if needed

---

## Security & Privacy

- All communication happens locally over USB
- No files are uploaded to any server
- No background access without user permission
